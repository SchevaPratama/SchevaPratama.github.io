About

Perkenalkan nama saya Scheva Fitrah Pratama nama panggilan saya sesuai orang orang di sekitar saya memanggil saya adalah Scheva.Alasan nama dari web ini adalah Zordiev bukan Scheva Pratama adalah karena Zordiev merupakan nama lain saya didunia maya/internet jadi saya ingin orang mengenal dua sifat dan dua nama di dua dunia yang berbeda.Saya lahir di Yogyakarata tanggal 18 Desember 2003 di Rumah Sakit katholik Bethesda pada malam Jum'at wage dengan keadaan normal.Agama saya islam walaupun saya lahir di rumah sakit katholik.Saya anak pertama dari dua bersaudara.Hobi saya bermain game walaupun memang tidak berguna tapi mungkin hanya itu yang saya suka lakukan saat ada waktu senggang,kadang juga ngoding jika sudah jenuh bermain game.Saya anak yang termasuk tidak bisa menjaga lisan,saya sering berkata kotor walau ditempat umum sekalipun.Saya tau itu salah namun itu sangat seru untuk dilakukan (Jangan Dicontoh).Jalan hidup saya hampir sama seperi hidup remaja yang lainnya tidak ada yang spesial,saya menjalani masa kecil yang cukup bahagia jadi saya tidak punya masa lalu yang kelam saya terlahir di keluarga yang bisa dibilang berkecukupan untuk memenuhi kebutuhan hidup.

Saya alumni dari SDN 2 Singotrunan,SMPN 1 Glagah dan sekarang saya (masih) sekolah di SMKN 1 Banyuwangi jurusan RPL maka dari itu saya membuat Web ini untuk tempat latihan untuk portofolio.Cita cita saya lumayan beragam,pada saat TK saya ingin menjadi pemadam kebakaran,Saat SD saya ingin menjadi astronot,pada saat SMP saya sadar saya tidak begitu pintar jadi saya tidak lagi bercita cita menjadi astronot lagi.Pada saat SMK pada awalnya saya ingin masuk ke jurusan TKJ karena saya mulai menyukai komputer saat SMP,tapi ibu saya menyarankan untuk masuk ke RPL karena beberapa alasan.Akhirnya saya masuk ke RPL dan semua tida seburuk yang saya pikirkan sebelumnya di RPL saya mulai mengenal beberapa bahasa pemrograman,saya mulai berpikir untuk menjadi seorang software developer tapi untuk sebagai cadangan saya belajar bahasa pemrograman yang lainnya mungkin sekian dari biografi yang bisa saya sampaikan karena saya masih muda dan belum memiliki cerita yang cukup banyak untuk diceritakan.Dan Terima Kasih Sebelumnya yang sudah mengunjungi web ini dan membaca biografi saya yang tidak penting ini.
